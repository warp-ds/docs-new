# Warp iOS release 0.0.69

## 2025-11-20

#### Logo component

 New Logo component with day-night mode support. Available also as icons with square diameter (use LogoSize = .small)

Supported brands: 
 - FINN
 - Blocket
 - Tori
 - DBA
 - Vend
 - Vend Pro
 - Autovex
 - Bilbasen
 - Qasa
 - Oikotie
 - Nettbil

 ```swift
Warp.BrandLogo.autovex(.default) // for standard logo size
Warp.BrandLogo.autovex(.small) // small size
```
